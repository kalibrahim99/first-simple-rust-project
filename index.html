What is Rust?

Rust is a modern systems programming language focused on safety, concurrency, and performance. It enforces memory safety at compile time using ownership and borrowing rules, while still producing fast native binaries.

When to use Rust

  • Systems programming (OS, drivers)
  • Performance-critical services and microservices
  • Command-line tools and developer tooling
  • WebAssembly (WASM) for browser or edge workloads
  • Where safe concurrency matters

Core features

  • Ownership, borrowing, and lifetimes (compile-time memory safety)
  • Zero-cost abstractions
  • Pattern matching and algebraic data types (enums)
  • Strong, static type system with type inference
  • Powerful tooling (Cargo, rustfmt, clippy)

Install (quick)

Official recommended way: rustup (works on Linux, macOS, Windows)

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

then restart shell or run

source $HOME/.cargo/env

Verify:

rustc --version cargo --version

Ownership (very short)

Every value has a single owner. When owner goes out of scope, value is dropped. Borrowing allows temporary access without transfer of ownership.

let s1 = String::from("hi");

let s2 = s1; // move: s1 is no longer usable // to borrow: let s3 = &s2; // s2 still owns the data

Example: simple function & enum

enum ResultState {
Ok(i32),
Err(&'static str),

}

fn add(a: i32, b: i32) -> ResultState { ResultState::Ok(a + b) }

fn main() { match add(2,3) { ResultState::Ok(n) => println!("sum = {}", n), ResultState::Err(e) => eprintln!("error: {}", e), } }
